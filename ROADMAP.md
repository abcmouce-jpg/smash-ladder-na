# Roadmap

A rough snapshot of where the project is headed — not a commitment or a schedule, just enough
for contributors to see what's planned, in progress, or shelved without having to guess.

## Now

- **Pre-season launch** — Saturday, July 25 at 6:00 PM ET. Site is live and open for signups;
  ranked play officially kicks off then.
- Building out community moderation (recruiting volunteers via Discord).

## Planned

- Richer local dev seed data (#17) — matches, a season, and tournaments, not just empty accounts.
- Integration tests for DB-dependent flows — lobby pairing, match confirmation, disputes (#18).
- Onboarding polish for new contributors (good first issues, clearer docs).

## Shelved / not planned right now

- Nothing formally shelved yet — this section will track anything considered and deliberately
  deprioritized.

## How this gets updated

This file isn't automated — it's a living doc. If you're picking up work that changes the
picture here, update this file as part of your PR. Open an issue first for anything non-trivial
(see [CONTRIBUTING.md](./CONTRIBUTING.md)) so the roadmap reflects agreed direction, not
just whoever got to it first.
